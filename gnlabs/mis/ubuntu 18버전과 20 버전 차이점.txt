ubuntu 18.04.5 버전과 20.04.3 다른점

차이점 1)
18 버전에는 python이 3.6 버전이 설치되어 있어서 python을 추가로 설치 하였고,
20 버전에는 python이 3.8 버전이 설치되어 있어 python을 설치 하지 않고 진행하였습니다.

차이점 2)
위의 차이에 따라 오류가 발생하여 따로 
$ sudo apt install gcc       
$ sudo apt-get install git
$ sudo apt install software-properties-common
명령어를 추가로 해주었습니다.

차이점 3)
마지막 예문 실행시에 2가지 버전 다 오류가 발생하였는데 numpy에 따른 오류로 
$ pip uninstall numpy
$ pip install numpy 
을 실행하였는데 mmdet3d가 지원하는 numpy<1.20.0 이라는 경고 문구가 뜹니다.
그래서 다시 
$ pip uninstall numpy
$ pip install numpy==1.19.0 
으로 1.19.0 버전을 설치 하였고, ubuntu 18버전에서는 실행이 되었고, ubuntu 20 버전에서는 실행이 되지 않았습니다.

경고를 무시하고 최신 버전으로 numpy를 설치하면 ubuntu 20 버전에서 실행이 됩니다. (다른 numpy 버전 1.19.0, 1.19.2, 1.19.5 버전도 다 안되었음)
